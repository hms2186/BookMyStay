# BookMyStay
Hemanth Madhav S-RA2411030010111
Book My Stay App is a Core Java project that demonstrates a scalable hotel booking system using real-world design principles.
It evolves through use cases covering OOP, data structures, and system architecture.
Key features include inventory management, FIFO booking, unique room allocation, and add-on services.
The project emphasizes type safety, modularity, and extensibility without breaking core logic.
It bridges the gap between academic concepts and production-level backend design.
